 ApexPlanet Data Analytics Internship - Task 2

📌 Project Overview
This project is part of the ApexPlanet Data Analytics Internship (Task 2). The objective is to analyze a sales dataset using SQL and build an interactive Power BI dashboard to uncover business insights and visualize sales performance.

 🎯 Objectives
- Perform SQL-based data analysis.
- Identify revenue trends and business insights.
- Build an interactive Power BI dashboard.
- Visualize sales performance across categories, cities, months, and products.

 🛠️ Tools & Technologies
- SQL
- Power BI
- Microsoft Excel
- CSV Dataset

 📊 SQL Analysis
The following analyses were performed:
- Total Revenue
- Top 5 Products by Revenue
- Revenue by Category
- Revenue by City
- Monthly Revenue Analysis
- Average Sales by Category

 📈 Dashboard Features
The Power BI dashboard includes:
- Total Revenue KPI
- Revenue by Month
- Revenue by Category
- Revenue by City
- Top 5 Products by Revenue

 📂 Repository Contents
- 📄 EDA.ipynb
- 📄 Task2_SQL_Queries.sql
- 📊 Task2_PowerBI.pbix
- 🖼️ Dashboard.png
- 📁 Cleaned_Sales_Dataset.csv

 🚀 Key Insights
- Identified the highest revenue-generating product categories.
- Compared sales performance across different cities.
- Analyzed monthly revenue trends.
- Determined the top-performing products based on revenue.
